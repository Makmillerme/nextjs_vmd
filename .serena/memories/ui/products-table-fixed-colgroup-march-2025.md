## Облік товару — таблиця без «вузького центру»

**Причина:** `table-auto` + `maxWidth: TABLE_COLUMN_MAX_WIDTH` (18rem) на кожній `th`/`td` обмежували ширину колонок; при 1–2 колонках сума ширин << контейнера → візуально сітка/фон виглядали як блок посередині.

**Виправлення** (`products-page.tsx`): `table-fixed`, `<colgroup>` з фіксованою шириною колонки «№» (`3.5rem`), решта `<col />` без width ділять залишок порівну; прибрано `maxWidth` з інлайн-стилів; «№» вирівняно `text-left` + `tabular-nums`; імпорт `TABLE_COLUMN_MAX_WIDTH` прибрано.
