Display → вкладка «Картка» (`display-page` → `TabsConfigManagement`): картка як у категорій — `rounded-lg border bg-card`, шапка `border-b bg-muted/30 rounded-t-lg` з `LayoutList`, `tabsListTitle`, кнопка Plus (ghost size-7) як у дерева категорій; пошук лише над карткою. Рядок як у категорій/StatusRow: `w-5` вирівнювання під шеврон; як у StatusRow: `w-5`; назва окремим `span` з `flex-1 min-w-0 truncate`; іконка як код — `font-mono text-xs` (sm); потім `Badge` defaultBadge; далі «N полів», стрілки, `#`; далі «N полів» (hidden sm:inline); стрілки лише якщо `!tab.isSystem`; `#` в кінці. У Sheet прибрано блок «Порядок» (дубль стрілок зі списку). список табів як у дерева категорій — зовнішній блок `rounded-lg border bg-card overflow-hidden`, внутрішній `divide-y divide-border` (без `TableWithPagination`). Рядок: стрілки вгору/вниз (stopPropagation), назва або `Badge` для системного табу (`SYSTEM_TAB_CONFIG.nameI18nKey`), іконка (sm+), `tFormat` `productsConfig.tabsConfig.tabsListFieldsCount` для кількості полів, глобальний `#` як `#{sortedTabs index + 1}`. Рядок: `justify-between` — ліворуч назва/бейдж + іконка лише якщо `tab.icon` не порожній (без «—»). Праворуч у групі: `tabsListFieldsCount` та `#`. Футер пагінації показується лише якщо `totalPages > 1`. Локаль `display.viewTable`: «Список» / List (режим прев'ю картки, не HTML-таблиця). Порожній стан: min-h центрований текст.
