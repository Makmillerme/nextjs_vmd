# Backlog — де зберігаються відкладені ідеї

**Шлях:** `nextjs_vmd/docs/backlog/ideas.md`

**Призначення:** Зберігати ідеї користувача, які відкладені на потім для доробок.

**Формат:** Markdown, кожна ідея — окремий блок з датою, суттю, статусом.

**Використання:** Додавати нові ідеї під час обговорення, коли користувач каже «відкласти» або «на потім».
