## Додавання поля в обрану секцію (ряд/колонка)

### Проблема
При кліку на "+" у порожній клітинці рядка 2, поле додавалось у рядок 1 (наступна вільна позиція за порядком).

### Рішення
1. **onClickAddField(tabId, row?, col?)** — передає позицію порожньої клітинки
2. **addFieldTarget** — зберігає { row, col } до вибору поля
3. **computeGridLayout** — підтримка targetRow/targetCol для розміщення в обрану позицію
4. **position-based placement** — order = row*3+col використовується для розміщення (таб-конфіг вже так зберігає)
5. **API** — order = targetRow * 3 + targetCol при додаванні
